## Funcionalidades

- Autenticação: login e registro com Google ou GitHub
- Editor em tempo real para criação e personalização do currículo
- Armazenamento no banco de dados dos currículos criados
- Geração de conteúdo com Inteligência Artificial (sugestões e otimizações)
- Funcionalidades premium com compra via Stripe
- Exportação para PDF a partir de componentes HTML, preservando o layout
